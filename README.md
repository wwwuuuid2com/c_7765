<h3>源码简介：</h3>
<p>小程序内含多个短视频去水印接口
支持全网几十款平台去水印和图集
另外该款小程序还带了一个强大的头像挂件制作功能
挂件制作内支持自动获取微信头像或者相册上传
里面有丰富的模板提供制作,包括节假日模板
去水印接口里面有一处小编个人感觉特别的强大
解析以后用户可以选择视频,或者多图还是文案模式
也支持修改短视频的MD5值
另外内还有壁纸和套图功能(自动采集)
另外小编还给添加了丰富的流量主模式
比如:激励视频,banner,插屏等
总得来说个人感觉还是可以的,具体大家自己研究一下
